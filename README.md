# EduBank
EduBank is a bank for schools to store data
\nLink: https://fbla-project.onrender.com/

## Sign up
When you arrive at the web page, you have to create an account. Hit the 'Get Started' button to be redirected to the sign up page. Fill out the data

## Logging in
After signing up, you should be automatically redirected to the log in page. Here, enter your username and password. If you arrive at the landing page, click the login button at the top to log into your account

## Logging out
If you're logged in, there should be a 'sign out' button at the top right of your screen. Click that to sign out. If you see a 'log in' button instead, that means you are logged out.

## Accessing the dashboard
To access the dashboard, you can press the purple 'dashboard' button at the top right of your screen. If that button isn't present and there's instead a 'log in' button, you'll have to log into your account first

## Using the dashboard
On the dashboard, you have 5 actions: 
- You can view the school database of partners, which gives you a table with all data.
- You can add new partners, which will bring up a form to input data about the new partner. Follow the instructions on the page. Afterward, you can view the new data by navigating to the database.
- You can get AI assistance by clicking on the AI/Help button. This assistant is powered by GPT-4.0. It has access to this readme file to assist you.
- You can delete data. If you're unable to delete data, it's normal. You have to be an administrator to delete data. If you're unable to, contact an adminstrator.
- You can modify data. If you're unable to modify data, it's also normal. You have to be an adminstrator to modify data. If you aren't, contact and administrator to do it for you.
